# IT Planning & Project Management - Assignments

## Description
A comprehensive collection of assignments and calculations from IT Planning and Project Management course. This repository contains project planning techniques, scheduling calculations, cost estimations, and resource management exercises.

## Topics Covered
- PERT (Program Evaluation and Review Technique) calculations
- CPM (Critical Path Method) analysis
- Project scheduling and time estimation
- Cost estimation and budgeting
- Resource allocation and planning
- Risk assessment and management
- Work Breakdown Structure (WBS)

## Technologies Used
- Microsoft Excel (calculations and charts)
- Project management formulas
